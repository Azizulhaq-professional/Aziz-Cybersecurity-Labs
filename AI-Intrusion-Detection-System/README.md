# AI Intrusion Detection System

## Overview
This project implements an AI-based Intrusion Detection System to identify malicious network traffic using machine learning techniques.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Google Colab

## Dataset
- NSL-KDD dataset

## Model Used
- Random Forest Classifier

## Workflow
1. Data loading and preprocessing
2. Feature encoding
3. Model training
4. Evaluation and accuracy measurement

## Results
The model successfully detects malicious activity with high accuracy, demonstrating how AI can support SOC teams.
## Real-World Use Case
This system can help security teams automatically detect intrusions and reduce response time.
